# 第1周：Python基础速成（面向Agent开发）

## 学习目标
掌握Agent开发中最常用的Python基础语法，不追求全面，只学高频出现的。

## 本周重点
1. 变量与数据类型（字符串、数字、布尔值）
2. 数据结构（列表、字典、集合）— Agent开发中最常用
3. 流程控制（if/for/while）
4. 文件读写（配置文件、日志记录）
5. 异常处理（API调用必备）

## 文件说明
- `basics.py`：基础语法演示代码，建议逐行阅读并运行
- `exercises.py`：10道练习题，带有 `# TODO` 标记，需要你补全代码
- `solutions.py`：参考答案，做完后再对照

## 学习建议
- 每天投入时间不限，但建议每天推进一点
- 先运行 `basics.py` 看效果，再动手做 `exercises.py`
- 本周额外任务：在LeetCode上完成至少10道"简单"难度的数组/字符串题目

## Agent开发场景映射
| 语法点 | Agent开发中的应用 |
|--------|------------------|
| 列表 | 存储对话历史消息 |
| 字典 | 存储API配置、模型参数、JSON响应 |
| 字符串格式化 | 拼接提示词（Prompt） |
| 文件读写 | 读取API密钥配置文件、记录日志 |
| 异常处理 | 处理API请求失败、网络超时 |
| 集合 | 关键词去重、数据过滤 |
